Our tools

Tools developed by researchers at ARIES LAB.

  • JNose Test

    JNose Test is a tool developed to automatically detect test smells in test code, and to collect coverage metrics.

  • RAIDE Plugin

    The RAIDE is an Eclipse plugin that automated detects and refactors test smells.

  • TSVizzEvolution

    TSVizzEvolution is a tool for visualizing the test smells and their evolution on software projects.

  • Lyzeli

    Speed up the qualitative and quantitative analysis of your research data, see charts, word cloud and more.

  • GrumPy

    The GrumPy Tool seeks to help researchers in order to facilitate the automated collection of Issues in GitHub's open-source repositories.
